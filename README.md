# Metropol-Theater Brunsbüttel Website

Ein modernes Webprojekt für das Metropol-Theater Brunsbüttel, entwickelt mit Tailwind CSS und Kirby CMS.

## Features

- Übersichtliche Darstellung von Inhalten
- Integration von Kirby CMS für einfache Inhaltsverwaltung
- Moderne UI-Komponenten mit Tailwind CSS
- Barrierfreie Gestaltung

## Technologien

- Tailwind CSS
- Kirby CMS
- PHP
